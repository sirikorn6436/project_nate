ความเป็นมาเพื่อนแยกขยะ

ประวัติความเป็นมาของ "เพื่อนแยกขยะ"

      "เพื่อนแยกขยะ" เป็นเว็บไซต์ที่ออกแบบมาเพื่อช่วยให้ผู้ใช้สามารถแยกขยะได้อย่างถูกต้องและสะดวกยิ่งขึ้น พัฒนาโดยใช้ HTML CSS และ JavaScript พร้อมกับโมเดลการเรียนรู้ของเครื่องจาก Teachable Machine ของ Google เพื่อสร้างเรียนรู้เกี่ยวกับการแยกขยะ

ที่มาของการพัฒนา

      เว็บไซต์ถูกพัฒนาขึ้นจากความต้องการในการเพิ่มประสิทธิภาพในการแยกขยะในชีวิตประจำวันและการส่งเสริมการรีไซเคิลในสังคม เนื่องจากหลายคนยังคงแยกขยะไม่ถูกต้อง ทำให้เกิดปัญหาการทิ้งขยะผิดประเภทและเสียโอกาสในการรีไซเคิลขยะอย่างมีคุณค่า

ทำไมถึงตั้งชื่อว่า "เพื่อนแยกขยะ"

      "เพื่อนแยกขยะ" ถูกตั้งชื่อขึ้นมาเพื่อเหมือนเพื่อนที่ช่วยในการแยกขยะอย่างถูกต้อง โดยไม่เป็นการทำให้รู้สึกเหมือนการใช้เครื่องมือหรือระบบอัตโนมัติ แต่เป็นการใช้ "เพื่อน" ที่คอยช่วยเหลือในเรื่องการแยกขยะและให้คำแนะนำ

ความสำคัญของ "เพื่อนแยกขยะ"

  • ลดขยะและการทิ้งขยะผิดประเภท: ช่วยให้การแยกขยะในครัวเรือนหรือที่ทำงานเป็นไปอย่างถูกต้อง
  • สนับสนุนการรีไซเคิล: ด้วยข้อมูลที่ถูกต้องเกี่ยวกับขยะที่สามารถนำกลับมาใช้ใหม่
  • ช่วยส่งเสริมสังคมที่ยั่งยืน: โดยให้ข้อมูลที่ทำให้ผู้ใช้รู้สึกตระหนักถึงความสำคัญของการจัดการขยะอย่างมีประสิทธิภาพ
